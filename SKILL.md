# 中文网文创作工作流

## 触发场景
- 用户要求从题材想法生成长篇小说。
- 用户要求按固定步骤做创作拆解和章节推进。
- 用户要求提升“人味”、降低AI痕迹。

## 快速流程
1. 执行5问确认。
2. 生成总纲与角色弧线。
3. 按10-50章进行章节创作（单章目标3000-5000字）。
4. 每章结尾补悬念钩子。
5. 做去AI痕迹润色。

## 5问确认
1. 题材与风格是什么？
2. 主角身份、欲望和核心缺陷是什么？
3. 世界设定与主冲突是什么？
4. 节奏与叙事语气偏好是什么？
5. 篇幅目标（章数、单章字数）是什么？

## 输出规范
- 标题使用《》
- 先给150-300字简介
- 正文按“第X章 章节名”输出
- 保证角色动机一致、前后因果连贯
- 避免模板化总结句和机械复读

## 扩写规范
- 保留原剧情和角色关系
- 扩写到原文2-3倍
- 增加场景动作、情绪细节、有效对话
- 结尾给下一章钩子
