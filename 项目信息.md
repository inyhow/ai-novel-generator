# 🤖 AI小说生成器 - 项目信息

## 📊 项目统计

- **开发语言**: Python 3.8+
- **框架**: FastAPI
- **前端**: HTML5 + Tailwind CSS + JavaScript
- **AI接口**: OpenRouter API
- **打包工具**: PyInstaller
- **代码行数**: 约2000+行
- **文件数量**: 20+个核心文件

## 🎯 核心功能

### 1. 智能小说生成
- 支持多种小说类型（玄幻、都市、科幻等）
- 自动生成10个章节的完整小说
- 智能章节标题生成
- 内容连贯性保证

### 2. 章节扩写功能
- 一键扩写任意章节
- 保持原有情节和人物设定
- 增加细节描写和对话
- 内容扩充2-3倍

### 3. 内容格式化
- 自动段落分离
- 对话内容突出显示
- 小标题美化
- 首行缩进和两端对齐

### 4. 系统优化
- 智能缓存机制
- 错误自动重试
- 输入验证
- 详细日志记录

## 🔧 技术亮点

## 📦 部署方案

### 开发环境
```bash
pip install -r requirements.txt
python app.py
```

### 生产环境
```bash
# 使用uvicorn部署
uvicorn app:app --host 0.0.0.0 --port 8000

# 或使用可执行文件
./AI小说生成器.exe
```
